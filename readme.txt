== Usage ==

Add the Percentage Increase Calculator shortcode to your page, post or sidebar:

`[ci_percentage_increase_calculator]`

Add the following code to your template where you would like the Percentage Increase Calculator to appear:

`<?php display_ci_percentage_increase_calculator(); ?>`
